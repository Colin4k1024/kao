<action>
1. Create optimized `backend/Dockerfile`:
   - Multi-stage build (builder → runtime)
   - Use alpine base image for runtime
   - Copy only necessary files
   - Use layer caching optimally
   - Add health check instruction
   - Run as non-root user
   - Add entrypoint script

2. Update `backend/Cargo.toml`:
   - Configure release profile with LTO and opt-level 3
   - Add features for production build
   - Optimize dependencies

3. Create `.dockerignore`:
   - Exclude unnecessary directories
   - Exclude development dependencies
   - Exclude documentation

Dockerfile structure:
```
# Build stage
FROM rust:1.70-alpine AS builder
# Build with optimizations

# Runtime stage
FROM alpine:latest AS runtime
# Install only runtime dependencies
COPY --from=builder /app/target/release/kao /usr/local/bin/kao
HEALTHCHECK --interval=30s --timeout=10s --start-period=5s \
  CMD wget --no-verbose --tries=1 --spider http://localhost:8080/api/health || exit 1
USER nonroot
ENTRYPOINT ["/usr/local/bin/kao"]
CMD []
```
</action>
